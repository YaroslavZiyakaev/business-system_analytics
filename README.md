# business-system_analytics
# Проекты
  **1.Анализ приемной кампании университета(MySQL)**
  
*О проекте:*

Данный проект реализован для **учебной базы данных университета**, которая содержит информацию о:
- факультетах и образовательных программах;
- абитуриентах, их достижениях и баллах ЕГЭ;
- правилах приема (минимальные баллы, план набора);
- поданных заявлениях.

*Задачи:*

1. Формирование списков абитуриентов: вывод кандидатов, подавших заявления на конкретную образовательную программу.
2. Расчет конкурса на программы: определение соотношения числа поданных заявлений к плану набора для оценки востребованности направлений.
3. Подсчет суммарных баллов ЕГЭ: вычисление итогового балла каждого абитуриента по предметам, необходимым для поступления на каждую программу.
4. Выявление непрошедших кандидатов: идентификация абитуриентов, чьи баллы по одному или нескольким предметам ниже установленного минимального порога.

*Используемые инструменты:*

- MySQL
- DBeaver
